# PD career source audit

검토일: 2026-07-15
원본: 포트폴리오 소유자가 제공한 34쪽 PD 포트폴리오 PDF

원본 PDF에는 전화번호, 이메일, 주소 등 개인정보가 포함되어 있어 공개 저장소에는 넣지 않는다. 이 문서는 공개 가능한 경력 사실과 링크만 남긴 검증 메모다.

## 공개 기준

- `84편+`는 PDF에서 편수가 명시된 피닉스박 `70편+`와 Pranky Friends `14편`의 합계다.
- 다니엘 프로젝트 작업은 편수가 명시되지 않아 `84편+`에 더하지 않았다.
- `2,000만+` 누적 조회는 PDF 원문에 적힌 당시 수치다. 현재 조회수를 다시 합산한 독립 검증 수치로 표현하지 않는다.
- Forbes Korea는 이혁수 개인의 수상이 아니라 Pranky Friends가 사회적 메시지 콘텐츠 사례로 소개된 근거로 사용한다.
- 공개 영상은 전체 제작물의 샘플이며 전체 제작 건수로 표현하지 않는다.
