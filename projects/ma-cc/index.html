MA-CC 코칭 & 컨설팅

Independent PM · Service Strategy & Delivery

컨설팅의 추상적인 가치를 신뢰 가능한 브랜드 경험과 문의 흐름으로 바꾸고, 기획부터 배포까지 단독으로 완성했습니다.

Live Site ↗ GitHub ↗

Contribution

브랜드·콘텐츠 구조, UI, 반응형 개발, 문의 백엔드, 콘텐츠 저장, 배포

Problem

전문성은 있었지만 웹에서 신뢰를 판단할 구조가 부족했습니다.

잠재 고객이 서비스의 차이와 진행 방식을 이해하고 바로 문의할 수 있도록 브랜드 서사와 전환 경로를 한 흐름으로 정리해야 했습니다.

  • 브랜드 메시지와 서비스 설명을 웹 정보 구조로 번역해야 했습니다.
  • 문의가 이메일까지 안정적으로 전달돼야 했습니다.
  • 운영자가 콘텐츠를 관리할 수 있어야 했습니다.

Decision

  • 서비스 설명과 문의를 하나의 서사로 연결 별도 브로슈어 없이도 방문자가 적합성을 판단하고 다음 행동을 선택하게 하기 위해서입니다.
  • Next.js API Routes + Nodemailer 외부 폼 서비스 의존 없이 문의 상태와 오류 처리를 제품 안에서 제어하기 위해서입니다.

Process

  1. 01
    브랜드 구조화

    핵심 고객, 서비스 가치, 신뢰 근거와 문의 전환 문장을 정리했습니다.

  2. 02
    디자인·구축

    반응형 UI, 모션, 콘텐츠 관리와 문의 API를 한 코드베이스에 구현했습니다.

  3. 03
    배포·운영

    실서비스 도메인에 배포하고 문의 흐름과 콘텐츠를 운영 가능한 상태로 전달했습니다.
